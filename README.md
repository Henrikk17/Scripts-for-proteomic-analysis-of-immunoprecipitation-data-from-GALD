# Scripts-for-proteomic-analysis-of-immunoprecipitation-data-from-GALD
This repo was made for sharing the code used for generating my master's thesis at DTU, which concerned analysing proteomic data (immunoprecipitation experiments) from Gestational alloimmune liver disease (GALD). The code is mostly unfunctional without the proteomic data. The proteomic data is private and can't be shared.
