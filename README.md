# Scripts for proteomic analysis of immunoprecipitation data from GALD
This repository contains the bioinformatic pipeline, data processing scripts, and visualization code developed for my Master’s thesis at the Technical University of Denmark (DTU). 
The scripts were designed to perform the analysis and plotting for the immunoprecipitation experiments targeting GALD-associated antigens.
<br>
**Data Access and Functionality**
**Restricted Data**: The raw proteomic datasets utilized in this analysis are private and contain sensitive clinical information; therefore, they are not included in this repository.
<br>
**Dependencies**: Consequently, these scripts are not functional without the original, proprietary data. The repository is intended to provide transparency regarding the methodology, statistical approach, and analytical workflow utilized in the thesis.
<br>
**Methodology and AI Collaboration**
The analytical code in this repository was developed in collaboration with AI-assisted programming tools (Gemini). All scripts have been manually reviewed, tested, and validated for scientific accuracy to ensure they meet the requirements of the thesis. Please note that some comments within the scripts may reflect AI-generated documentation.
