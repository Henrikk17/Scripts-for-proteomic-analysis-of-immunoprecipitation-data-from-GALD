# Scripts-for-proteomic-analysis-of-immunoprecipitation-data-from-GALD
This repo was made for sharing the code used for analysing proteomic data and generating the plots and figures for my master's thesis at DTU. 
Specifically, it concerns analysing proteomic data (immunoprecipitation experiments) from Gestational alloimmune liver disease (GALD). The code is mostly unfunctional without the proteomic data. The proteomic data is private and can't be shared.
